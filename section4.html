Computer File paths

There are two way to define a position of a file inside a computer:
  1. Absoulte path: Is the file path realtive to the root (e.g C:)
  2. Relative path: Is a file path realtive to the current work path. Most used for web-develpment since it is not important if you move the full project.
There are special characters to navigate
  1. .. : It goes to the precedent directory.
  2. .: It means from here, the current directory.
